글꼴 관련 속성

font-style

글씨체의 모양을 지정. oblique(기울임), italic(이텔릭) 등이 있음(동일하게 표현됨)

font-size

글씨체의 크기를 지정.
em : 1.5em
rem : 1.5rem

font-weight

글씨체의 두께를 지정. bold, lighter, bolder , 등이 있음

font-family

글꼴을 지정하는 속성
